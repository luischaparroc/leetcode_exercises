# Running Sum of 1d Array

Given an array `nums`. We define a running sum of an array as `running_sum[i] = sum(nums[0]...sums[i])`.

Return the running sum of `sums`.

## Example 1:

```
Input: nums = [1, 2, 3, 4]
Output: [1, 3, 6, 10]
```

## Example 2:

```
Input: nums = [1, 1, 1, 1, 1]
Output: [1, 2, 3, 4, 5]
```

## Example 3:

```
Input: nums = [3, 1, 2, 10, 1]
Output: [3, 4, 6, 16, 17]
```
