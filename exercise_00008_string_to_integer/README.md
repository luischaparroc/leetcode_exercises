# String to Integer (atoi)

Implement the `myAtoi(string s)` function, which converts a string to a 32-bit signed integer (similar to C/C++'s `atoi` function).

## Example 1:

```
Input: s = "42"
Output: 42
```

## Example 2:

```
Input: x = "     -42"
Output: -42
```

## Example 3:

```
Input: x = "4193 with words"
Output: 4193
```
